A coding agent in a bash loop, fed the same `PROMPT.md` until the project builds itself. Coined by Geoffrey Huntley in July 2025 and widely adopted across Claude Code, Codex, and Amp.

The mechanism is the simplest possible:

```bash
while :; do cat PROMPT.md | claude -p --dangerously-skip-permissions ; done
```

Three files do the work:

- `PROMPT.md` — the only instruction the agent reads
- `AGENTS.md` — operational guide kept under 60 lines
- `IMPLEMENTATION_PLAN.md` — task list the agent updates each iteration

State lives on disk. The loop has no memory of prior runs; the agent rediscovers context every cycle from the files. That structure is the source of Ralph's strength (deterministic, restartable) and its limit (no awareness of code that already exists).

**Use for** greenfield projects, prototypes, MVPs.

**Avoid in** legacy codebases — Huntley flatly: "There's no way in heck I would use Ralph in an existing code base."

Anthropic shipped an official `/ralph-loop` plugin in December 2025 that replaces the bash wrapper with a Stop-hook and a `--completion-promise` sentinel.
